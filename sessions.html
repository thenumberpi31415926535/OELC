We aim to create an supportive and encouraging English environment in a casual setting.


Below is a listing of online English speaking sessions for people who are using English as a second language. We use a Discord server as our online community. To join the server you will need to:
  1. Join Pi's Discord server
  2. Sign up for events
  3. Check your audio and mic settings
